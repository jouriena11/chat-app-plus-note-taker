## **Installation**
The following npm packages must be installed to run this application:
- bcrypt v5.1.0
- dotenv v16.0.3
- express v4.18.2
- lint v0.8.19
- mysql2 v3.1.2
- nodemon v2.0.20
- sequelize v6.29.0

The installations can be done conveniently by the running the following command line at the root directory: 
```
npm i
```
